Module food.py:
Я создал модуль food.py - для еды

В модуль добавлены функции:

spawn_food(snake_body) - создает еду в случайном месте, гарантирует что еда не появится на змейке

draw_food(screen) - отрисовывает еду на экране

check_food_eaten(snake_head) - проверяет, съела ли змейка еду

increase_score(points=1) - увеличивает счет игры

get_score() - возвращает текущий счет

get_food_position() - возвращает позицию еды (если она есть)

is_food_spawned() - проверяет, заспавнена ли еда

В config.py добавлена одна переменная:

score = 0 - глобальный счет игры

Как работает мой модуль:

Еда создается только там, где нет змейки. При съедении еды счет автоматически увеличивается. Модуль самостоятельно следит за состоянием еды (есть она на поле или нет).
